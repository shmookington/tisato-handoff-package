# Netlify Static Hosting

## Recommended For

Fast static hosting with simple form support.

## Steps

1. Create a Netlify account owned by the client.
2. Create a new site.
3. Upload the contents of `public-site/`.
4. Set the custom domain.
5. Enable HTTPS.
6. If using Netlify Forms, replace the booking form behavior before launch.

## Notes

The included booking demo uses browser storage. It should be replaced with Netlify Forms or a backend before production.
