# cPanel / Traditional Hosting

## Recommended For

Clients using GoDaddy, Namecheap, Bluehost, HostGator, or a similar file-manager style host.

## Steps

1. Open the hosting file manager.
2. Locate `public_html`.
3. Upload the contents of `public-site/`.
4. Confirm `index.html` is at the root of `public_html`.
5. Enable SSL from the hosting dashboard.
6. Test the site on desktop and mobile.

## Notes

cPanel hosting can serve the static site, but booking/admin/email automation usually requires PHP, a database, or a third-party form service.
